What it can do

Every feature an energy consultant needs — in an open-source, self-hosted web app.

PVGIS TMY Climate Data

Fetches 20+ year typical meteorological year data from the EU's PVGIS satellite database for any location on Earth. Falls back to Open-Meteo or clear-sky if unavailable.

PVsyst-equivalent Physics

Full hourly simulation chain: Perez anisotropic sky model, physical IAM, Faiman cell temperature, PVsyst one-diode SDM, and CEC Sandia inverter model.

15,000+ Module Database

Searchable CEC module database with full one-diode parameters. Also supports PVsyst .pan/.ond file import and a parametric quick-entry mode.

Orientation Optimizer

Sweeps the full tilt × azimuth grid and renders an interactive heatmap showing the annual yield for every orientation. Instantly highlights the global optimum.

Full Loss Waterfall

Configurable DC/AC loss chain: soiling, LID, mismatch, DC wiring, availability, AC wiring, and transformer. Visualized as a horizontal waterfall chart per IEC 61724.

CSV / JSON Export

Download monthly yield tables as CSV and the full simulation summary — parameters, results, loss waterfall, and data source — as machine-readable JSON.

Physics Stack

Every simulation stage uses industry-standard models, equivalent to PVsyst and SAM.

Stage Model pvlib function
Climate data PVGIS TMY (20+ yr satellite) pvlib.iotools.get_pvgis_tmy()
Solar position Ephemeris algorithm location.get_solarposition()
Sky diffuse Perez anisotropic sky model irradiance.get_total_irradiance(model='perez')
IAM Physical (AR glass coating) pvlib.iam.physical()
Cell temperature Faiman (PVsyst default) pvlib.temperature.faiman()
Electrical model PVsyst one-diode SDM calcparams_pvsyst() + singlediode()
Inverter CEC Sandia / PVWatts pvlib.inverter.sandia() / pvwatts()
Performance ratio IEC 61724 (vs POA) E_AC / (H_poa × P_peak)

Run Solar Advisor

Solar Advisor is hosted on Streamlit Cloud. Click below to launch it in this window, or open it in a dedicated browser tab. You can also run it locally with a single command.

  1. Clone the repository Get the source code from GitHub in one line.
  2. Create the conda environment All dependencies are pinned in solar-app/requirements.txt.
  3. Launch the app Streamlit starts a local server and opens your browser automatically.
bash — terminal
# 1. Clone git clone https://github.com/rgutzen/solarflower-app.git cd solarflower # 2. Create environment (conda / mamba) conda create -n app-dev python=3.12 conda activate app-dev pip install -r solar-app/requirements.txt # 3. Launch cd solar-app streamlit run app.py
